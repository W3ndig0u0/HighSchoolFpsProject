# Operation: High School 🔫🎯

A fast-paced **First-Person Shooter (FPS)** built in Unity. This project focuses on tight character controller mechanics, weapon handling, and level flow.

## 🎮 Play the Build
Check out the live version here:
**[Play on itch.io (yep2)](https://w3ndig0u0.itch.io)** 
*(Note: Link updated to reflect your itch profile)*

## ✨ Features
*   **Tactical FPS Controller**: Smooth movement, aiming (ADS), and recoil mechanics.
*   **Weapon System**: Hitscan/Projectile logic with reload animations and sound effects.
*   **Combat AI**: Basic enemy pathfinding and engagement behavior.
*   **Optimization**: Designed to run smoothly in-browser via WebGL.

## 🛠️ Tech Stack
*   **Engine**: Unity 3D
*   **Language**: C#
*   **Input**: Unity Starter Assets / Custom Input System
*   **Environment**: ProBuilder / Modular Assets

## 🚀 Installation & Setup
1.  **Clone the repo**:
    ```bash
    git clone https://github.com
    ```
2.  **Open in Unity**: Recommended version 2021.3+.
3.  **Play**: Open `MainScene` and press **Play** to test the mechanics.
